How this page works (read before using)

This page is a single static HTML file with no server of its own. When you add or edit data, changes are only saved in this browser (localStorage). For visitors elsewhere to see the new data too, after saving click "Download website file" below, then upload the new .html file to your hosting (replacing the old one). Back up your data (JSON) regularly to avoid losing it if the browser storage gets cleared.
